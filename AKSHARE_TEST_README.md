# AKShare 数据获取测试说明

## 问题描述

当前系统使用 AKShare 的 `stock_zh_a_hist` 接口获取A股历史数据,但遇到以下问题:

1. **网络连接失败**: 东方财富服务器返回 SSL 错误
2. **服务器限制**: 东方财富对高频请求实施访问限制
3. **数据异常**: 某些股票的后复权数据会出现负值

## 解决方案

### 方案1: 等待服务器限制解除
东方财富服务器的访问限制通常会在几小时后自动解除。

### 方案2: 使用代理
配置代理服务器:
```bash
export HTTP_PROXY=http://your-proxy:port
export HTTPS_PROXY=http://your-proxy:port
```

### 方案3: 切换网络环境
尝试使用移动网络热点或其他网络环境。

### 方案4: 使用本地数据
如果已有历史数据,可以跳过数据获取步骤,直接进行回测。

"
